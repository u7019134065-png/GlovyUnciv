# 🌍 GlovyUnciv

**Повний мод для Unciv — Нація Україна** 🇺🇦

## 📖 Про мод
GlovyUnciv додає детальну націю **Україна** з історичним лідером, реальними містами, унікальними діалогами та бонусами.

## ✨ Особливості нації
- **Лідер**: Volodymyr Zelenskyy
- **Унікальна здатність**: Unbreakable Spirit — бонуси до культури, оборони та сучасних юнітів
- **Стартовий bias**: Plains & Grassland
- **Кольори**: Жовтий та Синій
- **Міста**: Київ, Харків, Одеса, Львів та багато інших реальних українських міст

## 🚀 Встановлення
1. Завантажте цей репозиторій як ZIP.
2. Розпакуйте в папку `mods` гри Unciv.
3. Запустіть гру → Mods → Увімкніть **GlovyUnciv**.

Слава Україні! 💙💛
