# GlovyUnciv

**Мод для Unciv з нацією Україна 🇺🇦**

## Опис
Це мод додає повноцінну націю **Україна** до гри Unciv. З реальними містами, лідером, діалогами та унікальною здатністю.

## Особливості
- Лідер: Volodymyr Zelenskyy
- Унікальна здатність: Unbreakable Spirit
- Багато реальних українських міст
- Жовто-сині кольори

## Встановлення
1. Завантажте репозиторій як ZIP.
2. Розпакуйте папку `GlovyUnciv` в папку модів Unciv.
3. У грі увімкніть мод.

## Структура
- `jsons/mod.hjson` — основний файл мода
- `jsons/Nations.json` — нація Україна
