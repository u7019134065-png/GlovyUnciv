# GlovyUnciv 🇺🇦

**Повний мод для Unciv — Цивілізація Україна**

## Про мод
GlovyUnciv — якісний мод, який додає **Україну** як грабельну цивілізацію в Unciv.

### Ключові особливості:
- **Лідер**: Володимир Зеленський
- **Унікальна здатність**: "Незламний дух"
  - +20% Культури в прибережних містах
  - +15% Виробництва оборонних юнітів
  - Сучасні юніти сильніші
- **Стартовий баяс**: Рівнини та луки
- **Кольори**: Жовтий (#FFD700) та Синій (#0033A0)
- **Міста**: Київ, Харків, Одеса, Львів, Дніпро, Запоріжжя та понад 25 реальних українських міст

## Встановлення мода
1. Завантажте цей репозиторій (Code → Download ZIP)
2. Розпакуйте папку `GlovyUnciv` у директорію `mods` гри Unciv
3. У грі: **Additional mods** → Увімкніть GlovyUnciv

## Структура
- `mod.hjson` — основна інформація
- `jsons/Nations.json` — Україна
- `jsons/` — для розширення (Buildings, Units)
- `Images/` — для іконок (додайте пізніше)

## Розробка
Мод готовий до розширення. Додавайте унікальні юніти, будівлі, іконки!
